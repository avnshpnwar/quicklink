About QuickLink?
What is QuickLink ?
QuickLink is a web application which store all useful site required by most of developers. You can quickly visit any site in Test, Syst and Prod using quicklink.
What are QuickLink mode?
QuickLink run in two mode.
  1. Recent Site Mode: This mode store recent site visite by you. By default QuickLink will show you this page.
  2. All Site Mode:This mode store all site available in QuickLink.
You can switch between these mode using Blue buttons available in navbar.
What are QuickLink filters ?
You can easily filter site you required by using QuickLink filters. Currently QuickLink has these filters.
  • Country: Site filter based on country. e.g. India, Australia
  • Category:Site filter based on site type. e.g. eBanking, Phone Banking
  • Solution:Site filter based on solution type. e.g Google, Facebook
Apart from this their is search box, which can easily filter site.
Adding, Editing and Deleting sites.
How to add Site ?
Click on wheel button on right hand site of navbar. Choose option to add site, fill the form and submit it. You won't be able to add site if site name or URL already exist in QuickLink.
How to to edit or delete site?
Click on wheel button on right hand site of navbar. Choose option to edit site, this will show you all sites in tabular format. Filter the site you want to edit/delete. Once you find your site click on Edit or Delete button. You will be represented by a form.
QuickLink Tips and Tricks.
In QuickLink world refresh is bad
To reset filter you don't need to click refresh again and again. You can simpley use Reset button. Refresh will fetch data from server, while Reset will use data already present in DOM, and thus is more fast and cheap.
